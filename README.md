# Vintage Turntable Restoration and Documentation

Technical documentation and reverse engineering of a vintage record player.

This repository documents the disassembly, analysis, restoration and electrical reconstruction of a vintage turntable whose official documentation appears to be lost.

The goal is to preserve technical information about the device and make it freely accessible for restoration, research and historical preservation.

---

## Device Identification

Manufacturer: La Voix de Son Maître (His Masters Voice france), Pathé-Marconi  
Type: 332
Model: 53  
Country of manufacture: France  
Estimated decade: 1950  
Serial number: 269302
